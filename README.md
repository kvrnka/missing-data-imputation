# Missing Data Imputation

Репозиторий с кодом к курсовому проекту по исследованию методов восстановления пропущенных значений в табличных данных.

## О проекте

Цель работы: сравнить качество импутации при разных механизмах и долях пропусков, а также проверить, как выбранный метод импутации влияет на качество downstream-моделей.

В проекте рассматриваются:
- механизмы пропусков MCAR и MAR;
- доли пропусков 5%, 20%, 50%;
- методы импутации: Simple, KNN (несколько вариантов), MICE, MissForest;
- оценка как по прямым метрикам импутации, так и по метрикам ML-моделей после импутации.

## Данные

Используются два датасета:
- Adult Income (задача классификации, смешанные признаки);
- California Housing (задача регрессии, в основном числовые признаки).

Структура данных:
- data/raw: исходные данные;
- data/processed/<dataset>/ground_truth.csv: подготовленные эталонные версии без искусственных пропусков.

## Структура репозитория

- imputation.py: реализации методов импутации;
- ampute.py: генерация искусственных пропусков;
- experiment_runner.py: запуск базовых экспериментов импутации;
- notebooks/01_data_preprocessing.ipynb: предобработка;
- notebooks/02_eda.ipynb: разведочный анализ;
- notebooks/03_imputation_experiments.ipynb: эксперименты по качеству импутации;
- notebooks/04_models.ipynb: оценка downstream-моделей;
- notebooks/05_plots.ipynb: визуализация результатов.

## Метрики

Для качества импутации:
- NRMSE (числовые признаки);
- accuracy (категориальные признаки);
- время импутации.

Для downstream-оценки:
- регрессия: RMSE, R2;
- классификация: accuracy, F1.

## Быстрый старт

1. Клонировать репозиторий.
2. Создать виртуальное окружение.
3. Установить зависимости:

	pip install -r requirements.txt

4. Запускать ноутбуки по порядку из папки notebooks.

## Воспроизводимость

Эксперименты запускаются с фиксированными random_state и повторениями, чтобы получать устойчивые средние значения и стандартные отклонения метрик.

## Результаты

В ходе экспериментов формируются таблицы с итоговыми метриками и графики сравнения методов по датасетам, механизмам пропусков и долям missingness.
